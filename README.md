# An IoT Smart Doorbell System Using Camera and Ultrasonic Sensor for Real-Time Visitor Detection and Context-Aware Monitoring 
An IoT Smart Doorbell System Using Camera and Ultrasonic Sensor for Real-Time Visitor Detection and Context-Aware Monitoring
📌 Overview

The Smart Doorbell System is an IoT-based home security solution that integrates Arduino hardware with Python-based automation to provide secure, real-time visitor monitoring and remote access control.

This system detects visitor presence using an ultrasonic sensor, captures images via a USB camera, and sends them to the homeowner through Telegram for verification. Based on user approval, the system automatically unlocks and opens the door.

🚀 Features
🔍 Visitor Detection using Ultrasonic Sensor
📸 Real-time Image Capture via USB Camera
📲 Remote Access Control using Telegram Bot
🔓 Automated Door Unlock System
🔊 Voice Confirmation Module
🚨 Security Alerts (GSM, Buzzer, LED)
⏱️ Auto Door Close after fixed time
🔌 Reliable 12V Power Supply System

🧠 System Architecture

🔄 Workflow
Visitor approaches door
Ultrasonic sensor detects presence
Camera captures image
Image sent to user via Telegram
User verifies and sends approval
Door unlocks and opens
Door auto-closes after 20 seconds
Alerts triggered for unauthorized access
